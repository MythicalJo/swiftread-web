# Shared Code

This directory is reserved for code shared between the mobile and web platforms.

## Future Contents

- **types/** - Shared TypeScript types and interfaces
- **utils/** - Common utility functions
- **constants/** - Shared constants and configuration
- **hooks/** - Reusable React hooks (if applicable)

## Usage

Code placed here can be imported by both:
- `mobile/` - React Native mobile app
- `web/` - Progressive Web App

This helps maintain consistency and reduces code duplication across platforms.
